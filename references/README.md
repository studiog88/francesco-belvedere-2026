# References Pack

Use this folder to keep visual and directional assets for quick implementation passes.

## Suggested contents
- `screenshots/`: 3-6 site references.
- `figma-exports/`: hero, card, and typography captures.
- `moodboards/`: optional additional direction.
- `style-notes.md`: short written direction and do/don't rules.

This keeps design context in one place and avoids losing decisions across chat threads.
